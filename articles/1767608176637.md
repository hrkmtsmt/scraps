## Permission-Policy

Permission-Policyは、ブラウザの機能を制御するためにのレスポンスヘッダー。

例えば、Webアプリケーションがカメラやマイクの機能を拒否したいときに使用する。この場合次のようなレスポンスヘッダーを含めることでカメラとマイクの機能を拒否することができる。

```
Permission-Policy: camera=(),microphone=()
```

`()`は全拒否という意味で、同一オリジン含む全てのオリジンでカメラとマイクの機能を拒否する。つまり、`iframe`からのアクセスも拒否することになる。

特定のオリジンだけを許可したい場合は次のような表現になる。`self`は同一オリジンを意味する。

```
Permission-Policy: camera=(self https://example.com)
```
